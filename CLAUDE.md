# Purpose

PVE FCLU ("First-class Logical Unit") is a vendor-neutral Proxmox VE storage
framework providing first-class per-virtual-disk volume services over Fibre Channel block storage:
one array LUN per virtual disk, with advanced storage services (snapshots, clones,
QoS, replication) offloaded to the array. Per-vendor drivers (Hitachi, Dell, Pure
Storage, IBM, NetApp, …) plug into a shared generic core.

It generalizes the single-vendor `pve-HitachiBlockPlugin` (the reference
implementation) into a reusable multi-vendor framework.
